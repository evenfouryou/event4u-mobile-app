  2. Fix These Potential Issues                                                                                                                     
                                                                                                                                                    
  A. Progressive Number on Substitution                                                                                                             
                                                                                                                                                    
  When Sostituzione="S":                                                                                                                            
  - The ProgressivoRiepilogo should INCREMENT (use next number, not same as original)                                                               
  - This is because it's a NEW transmission replacing the old one                                                                                   
  - Update the logic so resend uses: originalProgressivo + 1                                                                                        
                                                                                                                                                    
  B. Date/Time Format (no separators)                                                                                                               
                                                                                                                                                    
  - Dates must be YYYYMMDD (e.g., 20260124, NOT 2026-01-24)                                                                                         
  - Times must be HHMMSS (e.g., 143022, NOT 14:30:22)                                                                                               
  - Check DataRiepilogo, DataGenerazioneRiepilogo, OraGenerazioneRiepilogo                                                                          
                                                                                                                                                    
  C. Element Order Inside                                                                                                                           
                                                                                                                                                    
  Elements MUST be in this exact order (DTD requirement):                                                                                           
  1. DenominazioneTitolareCA                                                                                                                        
  2. CFTitolareCA                                                                                                                                   
  3. CodiceSistemaCA                                                                                                                                
  4. DataRiepilogo                                                                                                                                  
  5. DataGenerazioneRiepilogo                                                                                                                       
  6. OraGenerazioneRiepilogo                                                                                                                        
  7. ProgressivoRiepilogo                                                                                                                           
                                                                                                                                                    
  D. File Naming for Substitution                                                                                                                   
                                                                                                                                                    
  For substitution, the filename progressive should also increment:                                                                                 
  - Original: RCA_2026_01_24_001.xsi.p7m                                                                                                            
  - Substitution: RCA_2026_01_24_002.xsi.p7m
